# CineGenie 🎬

CineGenie is an AI-powered movie discovery app that recommends films
based on genre, language, and year preferences.

It fetches movie data from the TMDB API and uses an AI model to explain
why the selected movies are good recommendations.

## Features

- AI-powered recommendation explanations
- Genre, language, and year filters
- Movie poster browsing
- Pagination (Next 5 titles)
- React + Express architecture

## Tech Stack

Frontend
- React
- Tailwind CSS

Backend
- Node.js
- Express

APIs
- TMDB Movie API
- Groq LLM API
- Google GenAI
