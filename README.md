# CineGenie 🎬

CineGenie is an **AI-powered movie discovery app** that recommends films based on **genre, language, and year preferences**.

The app fetches movie data from the **TMDB API** and uses **AI models** to explain why the selected movies are good recommendations.

It helps users discover new movies while also providing **AI-generated insights** about the recommendations.

---

## ✨ Key Features

- 🎬 Movie discovery based on **genre, language, and release year**
- 🤖 AI-powered explanations for movie recommendations
- 🖼️ Movie poster browsing interface
- 📄 Pagination system to explore **next 5 recommended movies**
- ⚡ Fast **React + Express full-stack architecture**
- 🌐 Integration with **multiple AI APIs**

## 🧠 How It Works

1. User selects **genre, language, and year filters**
2. Backend fetches movies from the **TMDB API**
3. The system sends selected movies to an **AI model**
4. AI generates **human-like explanations**
5. Frontend displays movies with **posters and AI insights**
